# Ted Ligety Skiing Analyzer

A web-based tool that analyzes your skiing technique by comparing it to Olympic champion Ted Ligety's form, providing personalized improvement tips.

## Features

- **Video Upload & Analysis** – Upload your skiing video to analyze your technique
- **Comparison Engine** – Compares your technique against Ted Ligety's professional form
- **Personalized Tips** – Get specific, actionable advice on how to improve your skiing
- **Easy to Use** – Simple, intuitive interface for skiers of all levels

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- A skiing video file to analyze

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/ted-ligety-skiing.git
   ```

2. Open `ted-ligety-skiing.html` in your web browser

3. Upload your skiing video and start analyzing!

## How to Use

1. Click the upload button and select your skiing video
2. The analyzer will compare your form to Ted Ligety's technique
3. Review the detailed feedback and improvement tips
4. Practice the suggested techniques and upload again to track progress

## Technologies Used

- **HTML** – Structure and layout
- **CSS** – Styling and responsive design
- **JavaScript** – Video analysis and comparison logic

# Quiz & SAT Test Generator

An intelligent quiz generator that creates custom academic quizzes and mock SAT tests with detailed performance analysis and personalized improvement recommendations.

## Features

- **Custom Quizzes** – Generate quizzes on any academic subject
- **Mock SAT Tests** – Full-length SAT practice tests with realistic formatting
- **Auto-Grading** – Instant feedback on your answers
- **Performance Analytics** – Detailed analysis showing your strengths and weaknesses
- **Improvement Tips** – Personalized study recommendations based on your results
- **Multiple Subjects** – Support for various academic disciplines

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- No additional installations required

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/quiz-artifact-v7.git
   ```

2. Open `quiz-artifact-v7.html` in your web browser

3. Select a subject and start testing!

## How to Use

1. **Create a Custom Quiz:**
   - Select an academic subject
   - Choose the number of questions
   - Take the quiz and submit your answers

2. **Take a Mock SAT:**
   - Select "Mock SAT Test" option
   - Complete the full test
   - Review detailed results and improvement areas

3. **Review Results:**
   - View your score and performance breakdown
   - Read personalized tips for improvement
   - Identify topics to study further

## Technologies Used

- **HTML** – Structure and layout
- **CSS** – Styling and responsive design
- **JavaScript** – Quiz generation, auto-grading, and analytics
